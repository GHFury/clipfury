USE OBS FOR RECORDING
Recommended for best quality. OBS must be open with Replay Buffer enabled.
How to set up OBS Replay Buffer:
1. Open OBS → Tools → WebSocket Server Settings → Enable
2. Set your password above (or leave blank)
3. In OBS → Output → Replay Buffer → Enable, set duration to match clip length above
4. Click Connect above
